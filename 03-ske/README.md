# Exercise 3: SKE

## Introduction

In this exercise, you will build a complete Kubernetes platform on STACKIT from the ground up.
You'll learn how infrastructure is provisioned using code, how to establish foundational platform components, and how to deploy applications into your cluster.

The exercise is divided into three sequential parts:

1. **Provision Infrastructure with Terraform**
2. **Install Platform Foundations**
3. **Deploy an Application**

By the end, you'll have a production-like setup with a working application accessible from the internet.

## Learning Goals

In this exercise, you will:

- Repeat your new Infrastructure as Code knowledge by deploying a a second infrastructure stack
- Learn how you can install "packages" into your SKE Cluster with Helm and manage their lifecycle
- Understand how infrastructure, platform, and application deployments could together in a real-world scenario
- Combine different managed STACKIT services (SKE, databases, observability)

## Getting Started

Follow the sub-exercises **in order**. Each one builds on the previous step:

1. Start with [Sub-exercise 1: Deploy Kubernetes Cluster with Terraform](./1_alloc/README.md)
2. Then continue to [Sub-exercise 2: Install Cluster Foundations](./2_basics/README.md)
3. Finally [Sub-exercise 3: Deploy Nextcloud Application](./3_nextcloud/README.md)
